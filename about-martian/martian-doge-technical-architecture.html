MARTIAN DOGE TECHNICAL ARCHITECTURE

The innovative arithmetic financial service platform created by MARTIAN DOGE will redefine the blockchain-based arithmetic ecology, and give the platform arithmetic and devices certain value liquidity through specific ways, and finally create a metaverse arithmetic financial service platform integrating arithmetic service, financial service, flow service and information service. As the circulation pass of the ecology, MARTIAN DOGE is committed to building a bridge for mutual communication among the metaverse ecology.
In order to improve the efficiency of collaboration and information transparency, MARTIAN DOGE uses blockchain as the underlying technical framework, public ledger records and information data sharing. The project of MARTIAN DOGE is completely open source to ensure the openness and allow any node to contribute. Any node can access and contribute at any time, anywhere.

Underlying Protocol

  • The underlying protocol for MARTIAN DOGE is the BSC network, a protocol and framework for building and connecting to Ethernet-compatible blockchain networks. Aggregate scalable solutions on Ether to support multi-chain Ether ecosystem.
  • MARTIAN DOGE Underlying Protocol： MARTIAN DOGE's underlying protocol is a Layer 2 scalability platform that enables fast, simple and secure off-chain transactions, not only for payment transactions but also for broad off-chain smart contracts.
Dedicated to the implementation of plasma MVP (Minimum Operating Plasma), the WalletConnect protocol and the Ethernet event alerting engine - Dagger make outstanding contributions. martian DOGE tokens are used for equity bets and participate in the proof consensus mechanism of the sidechain network.
MARTIAN DOGE's mission is to leverage decentralization, scalability, and security to drive large-scale adoption through a two-tier scaling solution, and to leverage sidechains for off-chain computing to achieve this, while using the Plasma framework and a decentralized PoS verifier network to ensure asset security.
MARTIAN DOGE strives to address the scalability and usability of the blockchain ecosystem while not compromising on decentralization and leveraging the existing development community and ecosystem.
With a more robust framework to facilitate higher transaction throughput, MARTIAN DOGE is currently providing blockchain developers with the resources needed to build world-class Dapps across multiple industries (e.g. decentralized finance, gaming, enterprise software, etc.).
MARTIAN DOGE aims to solve these problems by providing a Layer 2 scaling solution that enables scale by leveraging sidechains for offline computing while securing assets with the Plasma framework and decentralized Proof-of-Stake (PoS) validators. mARTIAN DOGE will be the developer's deployment and running decentralized applications MARTIAN DOGE will be the de facto platform for developers to deploy and run decentralized applications.
Functional Features Scalability: Fast and low-cost transactions on the sidechain and validated on the main chain; BSC is the first compatible base chain.
High throughput: Up to 10,000 TPS is already possible on a single sidechain of the test network; multiple sidechains will further help horizontal scaling.
User Experience: Smooth user experience and development abstraction layer from the main chain to the MARTIAN DOGE chain; mobile application and software development tools with WalletConnect support. Security: The operator of the MARTIAN DOGE chain is also an equity owner of the PoS system.
